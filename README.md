# High-Redshift Quasar Clustering Analysis

This repository contains computational pipelines and analysis tools for studying the clustering properties of high-redshift quasars using observational survey data and cosmological simulations.

The project focuses on sparse quasar populations at high redshift, where incompleteness corrections, survey selection effects, and low number statistics become important challenges for clustering measurements.

---

# Scientific Goals

The main objectives of this project include:

- Measuring quasar clustering statistics at high redshift
- Investigating halo mass estimates from clustering signals
- Studying incompleteness effects in sparse survey populations
- Constructing and analysing cosmological lightcones
- Comparing clustering behaviour across luminosity and redshift bins

These analyses help constrain:
- large-scale structure formation,
- dark matter halo occupation,
- and the evolution of massive black holes in the early Universe.
